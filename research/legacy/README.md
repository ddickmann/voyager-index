# Legacy Research Archive

This directory isolates legacy GEM/HNSW-era material from the shard-first
product surface.

It contains:

- archived Python runtime modules that are no longer part of the shipped package
- historical benchmark harnesses and result files
- legacy tests and maintainer scripts
- older research and guide documents kept for reference only

Nothing in this directory is part of the main public docs, package, or CI
story unless a specific contributor workflow opts into it.
