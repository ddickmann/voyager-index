# Research Areas

This tree is outside the main shard-first product surface.

Use it for:

- archived legacy benchmark suites, tests, and runtime modules under `research/legacy/`
- path-sensitive native research code such as `research/gem_index/`
- vendored or comparative research material such as `research/vendor/`

The supported public package, docs, CI, and benchmark story do not depend on
this tree for first-run usage.
