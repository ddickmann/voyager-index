# Internal Materials

This directory holds in-repo material that is useful for maintainers, audits,
release work, and historical validation, but is not part of the public product
docs.

Layout:

- `contracts/`: internal contract and boundary reference docs
- `plans/`: engineering plans and audit plans
- `memos/`: decision records and promotion notes
- `validation/`: validation-bundle index and evidence notes
- `verification/`: recovery and structural verification reference files
- `benchmarks/`: internal benchmark harness notes that back the public benchmark story
