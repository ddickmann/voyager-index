# Maintainer Benchmarks

These scripts are contributor-oriented benchmark harnesses that are useful for
deep diagnostics, hardware comparisons, or historical experiments.

For the supported OSS benchmark entrypoint, use `benchmarks/oss_reference_benchmark.py`
and the guidance in `docs/benchmarks.md`.

For the optional Latence graph lane, use
`tools/benchmarks/benchmark_latence_graph_quality.py` and the graph benchmark
guidance in `docs/benchmarks.md`.
